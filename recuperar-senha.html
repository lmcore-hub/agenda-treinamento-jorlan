Recuperação de acesso

Solicitar recuperação de senha

A solicitação será registrada no banco central. Um administrador ativo deverá aprovar e definir uma nova senha no painel.

  • Se todos os administradores perderam o acesso, a recuperação precisa ser feita pelo SQL no Supabase.
  • Este fluxo não envia e-mail automaticamente. Ele registra a solicitação para aprovação administrativa.
